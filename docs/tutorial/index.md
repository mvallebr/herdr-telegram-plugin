# Tutorial

This tutorial walks you through setting up herdr-telegram-plugin end to end.

**What you'll build**: a Telegram bot that lets you talk to your herdr agents from your phone — each agent in its own forum topic.

**Prerequisites**:
- [Node.js](https://nodejs.org/) 20+
- [herdr](https://github.com/mvallebr/herdr) installed and working
- A Telegram account

**Time**: ~10 minutes from scratch.

## Steps

1. [Create a Telegram Bot](/tutorial/create-bot) — get a bot token from @BotFather
2. [Install the Plugin](/tutorial/install) — clone, install, build
3. [Configure & Run](/tutorial/configure) — config.toml, start the daemon
4. [Pair & First Message](/tutorial/first-run) — authorize the bot, test it
5. [Daily Usage](/tutorial/daily-usage) — commands, tips, troubleshooting

Let's start with [Step 1 →](/tutorial/create-bot)
